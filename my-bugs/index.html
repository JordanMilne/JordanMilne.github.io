My Bugs

I usually only write about bugs that I think are particularly interesting or fun, and I don’t have time to write about everything.

Still, for my own reference and anybody who’s interested, here’s a non-exhaustive list of public advisories attributed to me, issues with pending fixes are not listed unless disclosure deadlines were not met by vendor:

2014

  • Reddit: Email header injection
  • Reddit: Email batch processing DoS
  • Reddit: Private submission leak via POST_share
  • Reddit: Private comment leak via GET_gold
  • Reddit: Title slug leak for private submissions via GET_oldinfo
  • Reddit: Embeddable content leaks via redditmedia.com
  • Reddit: Information leak via redirect before auth check
  • Reddit: Private / deleted comment leak
  • Reddit: 13x CSRF
  • Mozilla: Information disclosure with *FromPoint on iframes (Blog post)

2013

  • Reddit: Logout CSRF
  • Reddit: Friend/bannote CSRF
  • Reddit: Multi description CSRF
  • Reddit: Insufficient clickjacking prevention
  • Yahoo: Cross-domain theft on *.yahoo.com via hk.promotions.yahoo.com (Blog post)
  • Yahoo: XSS on edit.travel.yahoo.com
  • Reddit: Response splitting due to redirect to PATH_INFO and lack of newline sanitization
  • Reddit: Private subreddit info leak via search API
  • Reddit: User and private subreddit info leaks via private data in JS embeds
  • Ifixit: Lack of sanitization in parts of wiki template engine
  • RequestPolicy: Bypass of whitelist by wrapping underlying transport URI in jar: URI (Blog post)
